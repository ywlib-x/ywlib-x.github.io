Header: <ywlib>

This is the primal-header of ywlib-system.
You can access to all contents only if coding #include <ywlib> after #define _ywlib_.

In this header, some premitive typenames are defined and second-headers are included.
All the contents declared in <ywlib> and its sub-headers are surrounded by namespace yw.

Definitions

DeclarationDescription

using cat1;is cat-type whose size is 1 byte. (equals to char8_t)
using cat2;is cat-type whose size is 2 byte. (equals to char16_t)
using cat4;is cat-type whose size is 4 byte. (equals to char32_t)
